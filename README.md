# Tetris

## Building
To pull in dependencies:

    git submodule init
    git submodule update

To build:

    make (or `make debug`)

To install:
    make install (or not)

## Dependencies, Libraries
-libbsd
-libsqlite3 (3.8+)
-libncurses (Or compliant implementations)
-libpthread
